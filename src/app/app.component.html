Hobby Paint Conversion

This is a simple application to compare and convert between different brands and series of paints. The goal is to be able to inform hobbiests to be able to substitute paints from one brand for another.

This application is inspired from the original work of Redgrimm's Paint conversion and after seeing LastingN's fork that pulled the data into a JSON format. The data used to feed this application was parsed from the HTML in Redgrimm's original HTML.

How-To Use

The drop-down is used to select the base paint brand. The first column shows the various paints from the brand and the series from which they belong. You may use the filter text box to search for a specific paint color to narrow the rows in the grid. The checkboxes are for enabling the comparison paint brands.

Can I Help?

You sure can! I started this site to improve on a tool that was shown to me. I looked at it as a way to keep my Angular development skills sharp outside of my professional life. If you're a web developer and want to help add functionality or clean-up the site then please submit issues and pull-requests. If you don't do software develpment but are a painter you can still help. The site's value increases as more paints are added. The original data was taken as a given from Redgrimm. I do not know the extent to which it is correct. I certainly don't own all these paints. Maybe you do! Maybe you even have a color calibration and scanner tool! Adding this data to the site or correcting existing data will help everyone that uses the application.
